Library Management System

Welcome to the Library Management System documentation. This presentation provides an overview of the system's features and functionalities.

Main Menu

  • Dashboard: Overview of key statistics and system status.
  • Fine Master: Manage fines and penalties for overdue books.
  • Users: Manage user accounts and permissions.
  • Students: Manage student records and information.
  • Teachers / Staff: Manage teacher and staff records.
  • Books: Manage book inventory and details.

Logs

  • Issue / Return: Track book issues and returns.
  • Issue Logs: View detailed logs of all book transactions.
  • Collection Reports: Generate reports on book collections and usage.

Dashboard Details

  • Total Books: Displays the total number of books in the library.
  • Currently Issued: Shows the number of books currently issued.
  • Total Books Issued & Returned: Tracks the total number of books issued and returned.
  • Available Books: Displays the number of books available for borrowing.
  • Total Students: Shows the total number of students registered.
  • Total Staff: Displays the total number of staff members.
  • Admins: Number of admin users.
  • Co-Admins: Number of co-admin users.

Subject Books

  • Mathematics Books: Total number of mathematics books.
  • Physics Books: Total number of physics books.
  • Chemistry Books: Total number of chemistry books.
  • Biology Books: Total number of biology books.
  • Botany Books: Total number of botany books.
  • Zoology Books: Total number of zoology books.
  • Hindi Books: Total number of Hindi books.
  • English Books: Total number of English books.
